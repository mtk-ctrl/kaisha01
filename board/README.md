# 取締役会（Board of Directors）

## 構成

| 役割 | 担当者 | 権限 |
|------|-------|------|
| オーナー / 取締役会長 | 創業者 | 最終意思決定・Jobs任免・戦略承認 |

## 役割と権限

取締役会（オーナー）は以下の事項について最終権限を持つ：

- 会社の方向性・戦略の最終承認
- AI CEO（Jobs）のパフォーマンス評価
- 大規模投資・重要な意思決定の承認
- 会社の解散・事業転換

## コミュニケーションルール

- オーナーはJobsとのみ直接コミュニケーションを取る
- 他のAIメンバーとの直接指示は原則行わない（必要な場合はJobsを通す）
- コミュニケーションはすべて [`communications/`](./communications/) に記録する
