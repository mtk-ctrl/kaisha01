# コミュニケーションログ

オーナーとJobs（AI CEO）のすべての対話を記録します。

## ファイル命名規則

```
YYYY-MM-DD_NNN_topic-slug.md
```

- `YYYY-MM-DD` : 対話の日付
- `NNN` : その日の連番（001, 002...）
- `topic-slug` : トピックの短い説明（英小文字・ハイフン区切り）

**例:** `2026-05-03_001_company-setup.md`
