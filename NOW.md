### 開発ルール

| やること | やらないこと |
|---------|------------|
| `claude/*` ブランチに push → Actions 経由でデプロイ | `main` に直接 push |
| push 前に `npm run build` を通す | ビルドエラーのまま push |
| Secret 変更は GitHub Secrets だけ更新 | Vercel Dashboard を直接触る |
