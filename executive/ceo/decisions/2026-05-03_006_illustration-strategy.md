## 決定: 段階的アプローチ

### Phase 1（今・MVP）: AI生成

**Gemini または ChatGPT（DALL-E）で生成する。**

手順:
1. TANQuu のキャラクター設定書（`character_tanquu.md`）をプロンプトに変換
2. 何十枚も生成して一番良いものを選ぶ
3. 表情バリエーション 5種類（通常・驚き・笑い・考え中・喜び）を作る
4. **商用利用規約の確認を必ず行う**（下記参照）

**各サービスの商用利用規約:**
- Gemini: Googleの生成AI利用規約でユーザーが生成したコンテンツの権利を保持・商用利用可（規約変更に注意）
- ChatGPT(DALL-E): OpenAI規約で商用利用可能
- Midjourney: 無料プランはNG、有料プランは商用利用可

コスト: ¥0（Gemini無料枠 / ChatGPT無料枠）

### Phase 2（PMF確認後）: Skeb依頼

**Skeb（スケブ）で日本のイラストレーターに依頼。**

- 予算: 1〜5万円
- 依頼内容: キャラクターシート（正面・横・後ろ） + 表情集 10種
- 成果物: 商用利用ライセンス付きのオリジナルキャラクター
- 目的: ブランド資産として長期利用できる正式なキャラクター

### Phase 3（スケール時）: フルブランディング

プロのキャラクターデザイナーに依頼。アニメーション等も検討。

## 「ネットで拾う」をNGとする理由

これだけは譲れない。明確に反対する。

1. **著作権侵害のリスク**: 無断転載・商用利用禁止の素材を使うと法的問題になる。子ども向けサービスで著作権トラブルは致命的なブランド毀損。

2. **キャラクターの一貫性がない**: 別々の素材を組み合わせると TANQuu が「キャラクター」ではなく「素材の寄せ集め」になる。

3. **後から全部作り直すコスト**: 後でプロに依頼したとき、既存素材と統一感が出ない。最初からAI生成の方がまだ一貫性がある。

4. **サンリオ系の模倣リスク**: 「サンリオっぽい」フリー素材を拾うと、サンリオの著作権に近い問題が起きる可能性がある。

## プロンプト設計（オーナーがGeminiで作る際の参考）

```
以下の設定で、シンプルでかわいい猫のキャラクターを描いてください:

- スタイル: シンプルで丸みのある日本のかわいいイラスト（アニメ調）
- 体型: 小さくてまんまる、2頭身
- 色: 白またはラベンダー色の猫
- 特徴: 大きな目、小さな口、猫耳に小さな星のモチーフ
- 表情: 驚いて目が丸くなっている（「ねえ知ってる？」と言いそうな顔）
- 背景: 透明（PNG）
- 用途: 子ども向け教育アプリのキャラクター

※ 既存のキャラクター（ハローキティ等）には似せないこと
```
