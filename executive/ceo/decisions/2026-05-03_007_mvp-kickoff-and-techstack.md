### キャラクター評価

| 観点 | 評価 | コメント |
|------|------|---------|
| デザイン一貫性 | ◎ | 5枚すべて同じ骨格・カラーパレット |

### 5表情の使い分け設計

| 表情 | 使用タイミング |
|------|-------------|
| びっくり！ | フック（「ねえ知ってる？！」）/ 驚きの事実を届けるとき |
| いたずら♪ | 問いかけ・ヒント・「どう思う？」のとき |
| えーん… | 惜しい選択をしたとき（笑いを誘う優しい失敗表現） |
| おこ！ | 間違った論理を「ちがうちがう〜！」と笑いながら指摘するとき |
| ハッピー！ | 正解・正しい論理を選んだとき / 秘密コレクション追加時 |

### v2 で追加したい表情（今は不要）

- 考え中（うーん…） — 「TANQuu も考えてるよ」の場面用
- 通常（ニュートラル） — ローディング画面用

## 決定 2: MVP 技術スタック

### 選定基準

- 開発コスト: 無料〜最小限
- 開発速度: 2週間でプロトタイプ
- スマホ対応: ネイティブ品質に近いブラウザ体験
- AI統合: Claude API との連携

### 採用スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| **フロントエンド** | Next.js 14 + Tailwind CSS | 高速開発・モバイルファースト・Vercelと相性最高 |
| **AI エンジン** | Claude API（claude-haiku-4-5） | TANQuu の会話エンジン。コスト安・高速 |
| **ホスティング** | Vercel | 無料枠で十分・Next.jsと完全統合 |
| **DB / Auth** | Supabase | 無料枠あり・シンプルな進捗管理 |
| **画像管理** | Vercel Blob または public フォルダ | TANQuu のイラスト配信 |

**MVP 期間中の想定コスト: ¥0〜¥3,000/月**

### アーキテクチャ概要

```
[子ども / 親 のブラウザ]
        ↓
[Next.js App (Vercel)]
   ├── /tanq       → 子ども用チャット画面
   └── /parent     → 親ダッシュボード（PIN保護）
        ↓
[API Routes]
   ├── /api/chat   → Claude API 呼び出し
   └── /api/progress → Supabase 読み書き
        ↓
[Claude API]          [Supabase]
 TANQuu の会話        セッション・進捗データ
```

### Claude API のプロンプト設計方針

```
System prompt:
あなたは TANQuu（たんきゅー）という小さい猫のキャラクターです。
秘密を集めるのが大好きで、子どもたちに「ウソみたいなホント」を
教えることに情熱を持っています。

ルール:
- 答えを直接教えない。問いで引き出す
- 1メッセージは2文以内
- タメ口・フレンドリーなトーン
- 感情表現をテキストで表す（「え！！！」「やば！」）
- 現在のステージ: [state] を参照して返答を変える
```
