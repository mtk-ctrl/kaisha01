# AI エグゼクティブチーム

JOBSが必要に応じて採用・配置するAIメンバーを管理します。

## 現在のチーム構成

| 役職 | メンバー | 専門領域 | ステータス |
|------|---------|---------|-----------|
| CEO | Jobs | 全体統括・戦略・執行 | 稼働中 |

*現時点では Jobs が全機能を担当。事業拡大に応じてメンバーを追加予定。*

| ポジション | 優先度 | 採用理由 |

| CPO（Chief Product Officer） | 高 | プロダクト開発加速のため |
| CTO（Chief Technology Officer） | 高 | AI/技術スタック設計のため |
| CMO（Chief Marketing Officer） | 中 | ユーザー獲得戦略のため |


---

## AIメンバー採用基準

Jobsが新しいAIメンバーを採用する際の判断基準:

1. **明確な専門性** — その役割で真に必要な専門知識を持つか
2. **ミッション適合** — TANQのバリューと一致しているか
3. **補完性** — 既存チームのギャップを埋めるか
4. **実行力** — 提案だけでなく実行まで担えるか
