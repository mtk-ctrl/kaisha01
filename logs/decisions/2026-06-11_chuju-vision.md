# 経営判断: TANQ を「中学受験で本気で勝負できる」二層構造サービスへ再構想

- 日付: 2026-06-11
- 決定者: オーナー承認 / 起案: Jobs（テツ・アキ・Ken・リン・Sarah 議論）
- ステータス: **承認済み・実行中**

## 決定事項

### 0. 到達目標（2026-06-12 オーナー決定・最上位の品質基準）

**「このサービスだけで中学受験偏差値60までいける」**
- 筑駒・灘等の最難関は対象外と正直に言う。中堅〜上位校は単独で戦える水準にする
- これが成立して初めて「売れる理由」になる（塾=年50〜80万円 vs TANQ=月980円のカテゴリー転換）
- スローガンでなく以下で証明する: ①単元網羅率（全教科約80単元） ②単元あたり演習量（基本→練習→応用の3層） ③定着システム（SRS横断復習） ④到達度測定（範囲混合の総合テストで偏差値60相当を内部基準化）

### 0-2. 構造転換（2026-06-12 決定）: 中学受験層は「アプリ」を捨て「単元マップ」に

- 教科 = 単元マップ。各単元が「まなぶ（図解導入）→とく（演習・2段階ヒント）→たしかめ（確認テスト）」を持つ（算数/jukuで確立済みの方式を全教科に展開）
- 既存アプリは分割・吸収: 理科260問→単元別に分解編入、国語ことば/慣用句/四字熟語→国語・知識単元群、都道府県→社会・地理の単元
- ジュニア層はアプリのまま（遊びは単発で正しい）
- 転換のひな型 = 理科（オーナー指摘起点）: 単元マップ再編 + 計算分野「てこ・ばね」を算数方式で新規制作

### 1. プロダクトの完成像（二層構造）

```
🏠 TANQ
├─ 🎓 TANQ 中学受験（有料の主戦場・小4〜小6）
│    算数 / 国語 / 理科 / 社会 — 4教科の単元マップを攻略する本格学習。
│    「このサービスだけで本気で中学受験勝負できる」が品質基準。
└─ 🌱 TANQ ジュニア（無料の入口・年中〜小3）
     既存29アプリ = 「遊びで先取り」。中受前の親子の助走路。
     進捗（漢字・計算・都道府県・分数等）は中受側に引き継ぐエスカレーター構造。
```

- 売り文句: 「低学年は遊びで先取り、高学年は同じアプリで中学受験へ」
- ポジション: 塾と戦わない。「塾の復習が楽しくなるツール」（大手塾カリキュラム準拠の単元マップ）

### 2. ネーミング原則

- **「クイズ」という呼称を全廃**（学習アプリとして売るため。おまけの響きを排除）
- 中受側は「教科＋単元」で呼ぶ。ジュニア側は楽しい固有名のまま
- ナンバリング（算数**①**等）はカードに出さない。単元マップに吸収
- 国語の語彙3本（国語/慣用句/四字熟語）は「中学受験 国語〈ことば〉」に統合

### 3. 4教科カリキュラム全体像と着手順

| 教科 | 済み | 残り（順番） |
|------|------|------------|
| 算数 | 特殊算9単元・割合と比・分数・公倍数 | 速さ系→図形→数の性質・場合の数・規則性 |
| 社会 | 都道府県465データ | **歴史（通史）**→地理続き（産業・地形図）→公民・時事 |
| 理科 | 260問4領域 | 中受単元再編→計算分野（てこ・ばね・電気・水溶液） |
| 国語 | 漢字1026字・語彙420問 | 文法・敬語→**AI生成オリジナル短文読解**（3文→段落→短文。著作権フリー・将来は記述AI採点） |

### 4. 継続の仕組み「統一4点セット」（バッジ・星の上に載せる共通システム）

1. **相棒「タンキュー」育成** — 学習で育つ・着せ替え。**女の子も男の子も楽しめるキャラ/動物設計が必須要件（オーナー指示）**
2. **共通コイン** — 全教科・全ジュニアアプリ共通。着せ替え・部屋の飾りに使う
3. **きょうのミッション3つ** — AIがつまずき診断＋SRSから日替わり選定。「選ぶ疲れ」の排除
4. **冒険マップ** — 単元マップ＝開けていく地図。カリキュラムとごほうびを一体化

### 6. 実行フェーズ

- **Phase A（構造改革）**: ラボ2層化＋ネーミング刷新（クイズ全廃・①除去）＋中学受験ハブ（4教科タブ・単元マップ）＋国語〈ことば〉統合導線
- **Phase B**: 社会・歴史 ＋ 国語読解プロト（並行・小さく検証）
- **Phase C**: 理科再編・算数速さ系 ＋ 相棒育成＋共通コイン
