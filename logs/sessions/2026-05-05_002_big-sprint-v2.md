## Supabase 設定手順（オーナーへ）

1. https://app.supabase.com でプロジェクト作成（無料）
2. SQL Editor で `tanq-app/supabase-schema.sql` を貼り付けて実行
3. Project Settings > API から URL と anon key をコピー
4. Vercel Dashboard > TANQ プロジェクト > Environment Variables に追加：
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `SUPABASE_SERVICE_ROLE_KEY`
5. Vercel が自動再デプロイ → 完了！
