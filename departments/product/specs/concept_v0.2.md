## ターゲットの再定義

### プライマリーターゲット（小4）

**子ども**: 勉強に関心が薄い。テスト60点台。親に言われてやる感じ。  
**親**: 中学受験を意識し始めた。でも子どもが全然やる気を出さない。

このギャップこそがTANQの市場だ。  
勉強が好きな子は塾に行けばいい。勉強が嫌いな子の扉を開けるのがTANQ。

### セカンダリーターゲット（同じ状況の子ども全国に何百万人もいる）

「中受を考えているがやる気がない小4〜5年生」は日本全国で数百万人規模。  
自分の子どもで検証して、それを全国に届ける。

## コアコンセプト: 「秘密コレクション」

### フレーミングの変換

勉強嫌いな子どもが嫌うもの: **義務感、評価、失敗**  
勉強嫌いな子どもが好きなもの: **秘密、驚き、自分で気づく体験、短時間の達成感**

```
❌ 「今日は密度の勉強をしよう」
✓  「水に氷が浮く本当の理由、知りたい？大人も意外と知らないやつ」

❌ 「学習進捗: 3/16 ユニット完了」
✓  「秘密コレクション: ★★★（今日3つ集めた！）」

❌ 「今日のテーマは浮力です」
✓  「ねえ、タイタニック号はなんで沈んだか知ってる？ちょっと面白い話がある」
```

### 探究ループ（v0.2版）

```
① Hook（つかむ）         2〜3分
   ウソみたいなホントの問いで会話スタート
   選択肢ボタンで答える（タイプ不要）

② Explore（一緒に考える） 5〜7分
   AIが問いで引っ張る。答えは教えない
   「それってなんでだろう？」「じゃあこれはどう？」

③ あ、そっか（発見の瞬間） 1〜2分
   子どもが自分で概念に辿り着く
   AIが「そう！それが○○ってやつ」と名前をつける

④ これ、受験で出るんだけど（接続） 1〜2分
   「今発見したこと、中学受験でそのまま出るよ」
   達成感を演出しながら受験との橋を見せる

⑤ 証明してみる（2問） 2〜3分
   過去問ベースの問題を2問
   「さっきの発見が使えるはず」というフレーム

⑥ コレクションに追加！（保存） 30秒
   「秘密: 氷が水に浮く理由」が追加される
   視覚的な達成感
```

**合計: 12〜18分**（集中力の現実に合わせた設計）

## 3モードの設計

### Spark Mode（小4 / 勉強嫌いな子）

- **入口**: 「ウソみたいなホント」フォーマット
- **AIキャラクター**: 少し謎めいた探偵・研究者っぽい雰囲気
- **セッション長**: 12〜18分
- **文体**: タメ口に近い、フレンドリー

### Deep Mode（中1 / 高モチベーション）

- **入口**: 「これ知ってる？実は奥が深くて」フォーマット
- **AIキャラクター**: 研究室の先輩・知的な兄貴分
- **セッション長**: 20〜30分（本人が望むならもっと）
- **文体**: 少し大人寄り、対等な議論ができる設計
- **特徴**: 高校受験・大学受験・純粋な知的探究まで対応

### Wonder Mode（小1 / 白紙）

- **入口**: 「知ってる？」シンプルな問いかけ
- **AIキャラクター**: 優しいお兄さん・お姉さん
- **セッション長**: 8〜12分（集中力が短い）
- **文体**: やさしい言葉、ひらがな多め
- **特徴**: 概念の土台を作る。試験接続はまだしない

## TANQ English（将来ロードマップ）

理科でPMF確認後、英語モジュールを追加。

**目標の階段**: 英検5級 → 4級 → 3級

**探究フレーミングの例**:
```
「なんで "I am" って言うの？なんで "I is" じゃないの？」
「英語には "はい" と "いいえ" 以外に "はい" がある。何だと思う？」
「apple を英語圏の子どもは最初どうやって覚えるの？」
```

英語は反復が必要なため、比率を調整:
- 探究: 40%（WHYで動機づけ）
- 練習: 60%（繰り返しによる定着）
